# 프로젝트 소개 : 게시판

## 회원관리기능
- 회원, 프로필 관리(닉네임, 관심사, 프로필 사진 등)
- 회원가입 기능
- 로그인 기능
- 프로필 수정하기 기능

## 게시판 관련 기능
- 게시글 생성
- 게시글 1개 가져오기
- 게시글 수정하기
- 게시글 삭제하기
- 게시글 좋아요 기능
- 게시글 필터링(좋아요 누른 글, 내가 작성한 글)
- 게시글 각 기능마다 권한 설정

## 댓글 관련 기능
- 댓글 생성
- 댓글 가져오기
- 댓글 수정하기
- 댓글 삭제하기
- 게시글 가져올 때 댓글도 가져오게 만들기
